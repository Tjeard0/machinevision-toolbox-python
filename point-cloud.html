class machinevisiontoolbox.PointCloud.PointCloud(arg, image=None, colors=None, camera=None, depth_scale=1.0, **kwargs)[source]ÔÉÅ

Create new point cloud object

Parameters

  • arg (open3d.geometry.PointCloud, ndarray(3,N)) ‚Äì point cloud data

  • image (Image, optional) ‚Äì image used to create colored point cloud, defaults to None

  • colors (array_like(3), str, optional) ‚Äì color for points, defaults to None

  • camera (CentralCamera, optional) ‚Äì perspective camera model, defaults to None

  • depth_scale (float, optional) ‚Äì depth scale factor, defaults to 1.0

Raises

  • RuntimeError ‚Äì PointCloud class requires Open3D to be installed: pip install open3d

  • ValueError ‚Äì depth array and image must be same shape

  • ValueError ‚Äì bad arguments

This object wraps an Open3D open3d.geometry.PointCloud object. It can be created from:

  • an Open3D point cloud object

  • a depth image as a 2D array

  • an RGBD image as a 2D depth array and a color Image. Camera intrinsics can be provided by a CentralCamera instance.

Seealso

open3d.geometry.PointCloud Image CentralCamera
