machinevisiontoolbox.Image.isrgb

property Image.isrgb

Image has RGB color order?

Returns

Image has RGB color order

Return type

bool

Example:

>>> from machinevisiontoolbox import Image
>>> img = Image.Read('flowers1.png')
>>> img.isrgb
True
Note

Is False if image is not color.
