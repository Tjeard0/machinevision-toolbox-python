Match SIFT point features

Parameters

  • other (SuperFeature2D) – set of feature points

  • ratio (float, optional) – parameter for Lowe’s ratio test, defaults to 0.75

  • crosscheck (bool, optional) – perform left-right cross check, defaults to False

  • metric (str, optional) – distance metric, ‘L1’, ‘L2’ [default], ‘hamming’, ‘hamming2’
